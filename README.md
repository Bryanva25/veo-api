## 🔧 How the Software Works

veo-api connects to Google Veo’s video generation service. It sends data like text or images, then gets back a video file created from that data. You use a simple window to input your text or select images. Behind the scenes, the API makes the video for you. The software manages the connection so you do not need technical knowledge.

## 📂 File Types Supported

- Text inputs: plain text files (.txt) or typing directly.
- Images: JPEG (.jpg, .jpeg), PNG (.png), BMP (.bmp).
- Video output: usually MP4, compatible with most players.
